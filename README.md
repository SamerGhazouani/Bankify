# Bankify (Online Banking Application)

This project is an online banking application designed to manage all aspects of modern banking. It provides a complete system for users to handle accounts, transactions, credit, reimbursements, and other banking services securely and efficiently.

# Features

- User Management: Create, update, and manage user accounts

- Accounts & Packs: Handle different types of accounts and banking packages

- Transactions & Transfers: Perform money transfers, view transaction history

- Credit & Reimbursement: Manage loans, credit applications, and repayments

- Cheques: Issue and manage cheques

- Branches & Insurance: Manage bank branches and insurance services

- Complaints/Reclamations: Submit and track complaints or requests

- Security: Secure authentication and data management
